Download, Unzip, Open and Personalizing the Template

링크

github.com/spctrm404/IMD232-Template

공통

  1. 위 링크에 접속.
  2. Code -> Download ZIP
  3. 압축해제. IMD232-Template 폴더 바로 아래 example.html 이 있도록 압축해제할 것.
  4. IMD232-Template 폴더를 IMD232_학번 으로 이름 변경.
  5. 이름이 변경된 IMD232-Template 폴더 (이하 템플릿 폴더)를 클라우드로 동기화관리 되지 않고 있는 폴더 아래로 이동.
    • 바탕화면을 동기화하는 경우가 많으니 주의.
  6. vscode를 실행한 뒤, 템플릿 폴더를 vscode에 드래그앤드랍
  7. example.html 을 index.html 로 이름 변경. (단축키: f2 | return )
