homebrew 설치
  1. brew.sh 에서 인스톨 커맨드를 복사.
  2. 터미널에 커맨드를 입력: 붙여넣고 엔터 = 설치시작.
  3. 설치과정은 알아서 잘.
  4. 설치 완료 후 터미널에 brew --version 이라 입력했을 때, zsh: command not found: brew 와 유사한 에러 문구가 뜨면, 아래 단계로, 그렇지 않다면 아래 단계를 무시하고 git 설치로.
  5. 터미널의 내용 중 
    ==> Next Steps:
Run these two commands in tour terminal to add Homebrew to your PATH:
(echo; echo 'eval "$(/opt/homebrew/bin/brew shellenv)"') >> /Users/사용자명/.zprofile
eval "$(opt/homebrew/bin/brew shellenv)"
    를 찾아 3번째 라인을 복사, 입력. 4번째 라인을 복사, 입력. 위의 코드를 복사-입력하라는 것이 아니다. 여러분의 터미널에서 위 내용을 찾아 복사-입력하라는 것이다.
  6. 다시 brew --version 이라 입력했을 때, zsh: command not found: brew 와 유사한 에러 문구가 안뜨면 설치가 완료된 것이다. 에러 문구가 뜨면? 몰?루?
