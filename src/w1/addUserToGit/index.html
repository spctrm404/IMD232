링크

GitHub - Email Settings

공통

  1. 위 링크: GitHub의 Email Settings에 접속. 로그인을 요구하면 로그인한다.
  2. Primary email address 혹은 Keep my email addresses private 을 에 적혀있는 숫자+본인아이디@users.noreply.github.com 를 찾아 복사해둔다. 이를 깃허브이메일 이라 부르겠다.
  3. Your personal account 라 적혀있는 바로 위 괄호로 감싸진 글자: 본인의 아이디를 찾아 복사해둔다. 이를 깃허브아이디 라 부르겠다.
  4. 위 두 과정에서 앞 뒤로 띄어쓰기가 포함되어 복사-붙여넣기되지 않도록 주의한다.

WIN

  1. 시작에서 git cmd 를 타이핑해 git bash를 찾아 실행한다.

MAC OS

  1. 터미널을 실행한다.

공통

  1. git bash 혹은 터미널에 다음과 같이 입력한다.
    git config --global user.name "깃허브아이디"
  2. git bash 혹은 터미널에 다음과 같이 입력한다.
    git config --global user.email 깃허브이메일
  3. git bash 혹은 터미널에 다음과 같이 입력했을 때, user.name에 깃허브아이디, user.email에 깃허브이메일이 보인다면 제대로 입력된 것이다.
    git config --list
