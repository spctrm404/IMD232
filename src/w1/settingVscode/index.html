Setting VSCode

settings.json 수정

  1. 좌측 하단그룹의 Manage (톱니바퀴 아이콘) -> settings (단축키: ctrl | cmd + , )
  2. 열린 setting 탭의 우측 상단의 문서모양 아이콘 (Open Settings (JSON)) 클릭
  3. 내부 내용을 아래 첨부한 settings.json 의 내용으로 덮어쓰고 저장 (단축키: ctrl/cmd + s )한다.

extension 설치

  1. 좌측 상단그룹의 Extensions (네개의 사각형 아이콘) (단축키: ctrl | cmd + shift + x )
  2. 이하의 검색어들에 대한 검색결과에서 install 버튼을 눌러 설치
    • stylelint.vscode-stylelint
    • standard.vscode-standard
    • cipchk.cssrem
    • esbenp.prettier-vscode
    • ritwickdey.LiveServer
    • glenn2223.live-sass
    • bradgashler.htmltagwrap
    • dbaeumer.vscode-eslint
    • emanuelebartolesi.css-initial-style-value
    • samplavigne.p5-vscode
